# Artwork-shopify-store-case-study

## 📌 Project Title

**Artist Portfolio & Shopify E-commerce Store – Custom Theme Development**

## 🧾 Overview

This project is a custom **Shopify eCommerce website** developed for an independent artist who creates handmade artwork, coloring books, and baseball-inspired illustrations. Before launching the website, the client primarily relied on Instagram to showcase and sell products, which made it difficult to present the full collection professionally and scale the business.

The main objective of this project was to create a visually engaging Shopify store that reflected the artist's unique style while providing a seamless shopping experience. The website was designed to showcase artwork collections professionally, strengthen the client's online brand presence, and increase product sales through a dedicated eCommerce platform.
